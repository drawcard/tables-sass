Basic tables

For simple table styling, use the .table class.

First Name Last Name Super Hero
Peter Parker Spiderman
Bruce Wayne Batman
Clark Kent Superman 
<table class="table">
    <thead>
        <tr>
            <th>First Name</th>
            <th>Last Name</th>
            <th>Super Hero</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Peter</td>
            <td>Parker</td>
            <td>Spiderman</td>
        </tr>
        <tr>
            <td>Bruce</td>
            <td>Wayne</td>
            <td>Batman</td>
        </tr>
        <tr>
            <td>Clark</td>
            <td>Kent</td>
            <td>Superman</td>
        </tr>
    </tbody>
</table>

Condensed Table

Reduce cell padding by adding the .table-condensed class.

You still need to include the .table class.

First Name Last Name Super Hero
Peter Parker Spiderman
Bruce Wayne Batman
Clark Kent Superman 
<table class="table table-condensed">
    ...
</table>

Bordered Table

Add borders and rounded corners by adding the .table-bordered class.

You still need to include the .table class.

First Name Last Name Super Hero
Peter Parker Spiderman
Bruce Wayne Batman
Clark Kent Superman 
<table class="table table-bordered">
    ...
</table>

Combining Styles

You can combine the .table-condensed and .table-bordered classes for additional styling.
